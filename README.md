# my-crystal-dish
My Crystal Dish


A dish control center for a restaurant. Where the manager can add and edit the available dishes. And customers can follow the lists of dishes.

---

## Docker

> Pull the image [my-crystal-dish-api](https://hub.docker.com/r/paulofsr/my-crystal-dish-api) from Docker Hub

```shell
$ docker pull paulofsr/my-crystal-dish-api:latest
```

> Pull the image [my-crystal-dish-web](https://hub.docker.com/r/paulofsr/my-crystal-dish-web) from Docker Hub

```shell
$ docker pull paulofsr/my-crystal-dish-web:latest
```
